# Instagram Manual QA Testing

## Project Overview

This project presents a comprehensive manual QA testing report for the Instagram mobile application. The testing focused on validating core functionalities, evaluating the user interface, assessing usability, performing exploratory testing, and documenting observed defects.

---

## Project Highlights

- 46 Test Scenarios
- 46 Detailed Test Cases
- UI Findings
- Usability Findings
- Exploratory Testing
- Bug Report
- Test Execution Summary

---

## Scope of Testing

The testing covered the following modules:

- Login
- Registration
- Home Feed
- Search
- Reels
- Profile
- Notifications
- Settings
- Logout
