#  Swiggy Manual QA Testing

## Project Overview

This project presents a comprehensive manual QA testing report for the Swiggy mobile application. The testing focused on validating key user workflows, verifying functional requirements, evaluating the user interface, assessing usability, and performing exploratory testing.

---

## Project Highlights

- 56 Test Scenarios
- 56 Detailed Test Cases
- UI Findings
- Usability Findings
- Exploratory Testing
- Test Execution Summary

---

## Scope of Testing

The testing covered the following modules:

- Authentication
- Location Selection
- Restaurant Browsing
- Search
- Cart Management
- Checkout
- Payment Options
- Order Tracking
- Notifications
- Profile
- Settings
- Logout
