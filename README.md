# Manual QA Testing Portfolio

Welcome to my **Manual QA Testing Portfolio**!

This repository showcases hands-on **Manual Quality Assurance (QA)** testing projects performed on real-world mobile applications. Each project demonstrates my approach to functional testing, test case design, UI validation, usability evaluation, exploratory testing, and defect reporting using structured QA documentation.

The objective of this portfolio is to demonstrate practical manual testing skills through well-documented test reports that reflect real application behavior and industry-standard testing practices.
